# AI-Enhanced Student Performance Analyzer

## Features
- XGBoost-based AI risk prediction
- Interactive Streamlit dashboard
- Student performance analytics
- Automated email integration
- Separate figure generation (5.1–5.7)

## Run Instructions

Install dependencies:
pip install -r requirements.txt

Run Analyzer:
python student_analyzer_enhanced_with_figure.py

Run Dashboard:
streamlit run student_dashboard.py
